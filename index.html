Pocket Rascal
- open in new tab -

In the spirit of Neopets and Tamagotchi, Pocket Rascal is your virtual, pocket lint friend to take care of, play games with, and customize. Keep your rascal happy and stylize them to your liking by unlocking new body styles, face components, and accessories through minigame challenges.

This application was built using React.js. The front end incorporates the CSS framework Material UI as well as the physics engine Matter.js. All animations were created via JavaScript and sprite sheets. User information is stored in a MySQL database with all sensitive information encrypted.

Have fun!

Wacky-Wiki
- open in new tab -

Wacky-Wiki is a modern twist of the nostalgic 90's activity 'Mad Libs'. Like Mad Libs, the user inputs words by parts of speech, and these words are injected into an article. Unlike Mad Libs, the user can input any topic, yielding endless possibilities of wacky results.

This application harnesses the power of Wikipedia's API, with which the user can search any topic and yield the same results they would get from Wikipedia's search engine. The UI was created mostly by raw CSS with a little help from Bootstrap. Data from this site is saved to local storage, allowing the user pick up where they left off.

Get wacky!

MetaMemeMarket

On MetaMemeMarket, users can buy/sell shares of their favorite memes using an in-app currency. Watch as your portfolio's value changes along with current meme trends, and upload original content to see how your memes thrive (or not) in an ever-changing economy.

Technologies used for this application include Node.js, Express.js, and Handlebars, as well as a MySQL database for user data. All sensitive information is encrypted using bcrypt. Images are hosted via Cloudinary, a third-party image hosting service.

Music/SFX/Dialogue demo

"AngryBots" was given to me as a soundless Unity project.

I composed a vertical score and created sound effects using a mix of digital synths, home-recorded foley, and audio samples from online collections. I borrowed the help of two 'voice actors' for dialogue.

Using Wwise, I compiled all audio elements into the game engine.

All instruments are virtual from my expansive VST collection. I created a five-layer vertical score and mixed it alongside gameplay footage to simulate the adding/subtracting of the layers in response to the fluctuating intensity of gameplay.

"Feather" 4-String Bass

❮ ❯
❮ ❯
This build was inspired by the work of Luthier Jens Ritter, as well as the musical styles of Les Claypool and Justin Chancellor.
❮ ❯
It features an alder body and quilted maple top. The neck is maple and the fingerboard is African wenge. Together, this wood combination gives this instrument a tight, punchy tone - rich in upper mids without sacrificing low end roundness. Versatility was the goal.
❮ ❯
The bass is fitted with a custom brass ferrule block to make up for any lost sustain due to the instrument's light-weight build.
❮ ❯
The onboard electronics include Bartolini Soapbar pickups and the Bartolini HR-5.2AP/918 3-Band NTMB+F Preamp Module. This system gives the bass a wide tonal range, while retaining the natural characteristics of tonewoods.
❮ ❯
All of the hardware is Hipshot.
❮ ❯
I hand-carved the neck to a thin profile with a shallow radius for ease and comfort of playability.
❮ ❯
The body shape is equal parts form and function. Ideal for seated playing, this bass is a great option for studio sessions.

"Ma'at" Acoustic Guitar

❮ ❯
❮ ❯
My second acoustic build, this guitar was built with articulacy and expressivity in mind.
❮ ❯
I chose to use Central American bocote for back and sides, and spruce for the soundboard. This tonewood combination as well as the OM body shape give this guitar a bright, sparkly tone with clear high frequency definition.
❮ ❯
The sides were bent using water, heat, and clamping pressure from a highly specialized jig.
❮ ❯
The soundboard is braced in the standard x-bracing style with scalloped tone bars. These tone bars provide structural support and enhance higher frequencies.
❮ ❯
The neck is mahogany, hand-carved to an oval 'C' shape.
❮ ❯
The bridge was also hand-carved. Like the fretboard, it is ebony, which improves the guitar's presence.
❮ ❯
This guitar features Mother of Pearl fretmarkers and a 12th-fret inlay of my insignia.
❮ ❯
I've heard talk that nitrocellulose lacquer provides significantly better tonal results than other guitar finishes. I think that's a bunch of mumbo jumbo but I used it anyways, just in case.
❮ ❯
There's nothing quite like building guitars in the warm Arizona sun.
